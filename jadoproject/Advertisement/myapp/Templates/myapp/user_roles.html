👥 User Roles & Permissions

Regular User

Standard user account with basic platform access

Permissions:

  • ✓ View and browse all advertisements
  • ✓ Create and manage personal ads
  • ✓ Like and comment on ads
  • ✓ Share advertisements
  • ✓ Send/receive private messages
  • ✓ Access personal dashboard
  • ✓ View share history

Administrator

Full system access for platform management

All Regular User Permissions +

  • ✓ Access admin dashboard
  • ✓ Create and manage advertisements
  • ✓ View and manage all users
  • ✓ Moderate advertisements
  • ✓ Review and handle reports
  • ✓ View analytics and statistics
  • ✓ System administration
  • ✓ Content moderation

Security & Access Control

  • Regular users cannot access admin functions or sensitive data
  • Admin privileges are granted only to authorized personnel
  • All access attempts are logged for security monitoring
  • Attempting to bypass restrictions may result in account suspension

Need Admin Access?

If you require administrator privileges for your role, please contact the platform administrator or site owner. Admin accounts are not available through regular signup.
